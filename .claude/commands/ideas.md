# Brainstorm Blog Post Ideas

Generate new blog post ideas based on trends and existing content.

## Instructions

1. Analyze existing posts in `_posts/`:
   - Identify main topics covered
   - Find gaps in content
   - Note successful post patterns

2. Research current trends:
   - Use web search for trending topics in James's areas of expertise
   - Look for recent developments in WebXR, Serverless, AWS, React Three Fiber

3. Generate 5-10 post ideas with:
   - Title
   - Brief description (2-3 sentences)
   - Target audience
   - Estimated complexity (beginner/intermediate/advanced)
   - Related existing posts that could link to it

4. Prioritize ideas by:
   - Relevance to audience
   - SEO potential
   - James's expertise fit
   - Content gaps being filled

## Focus Areas

- WebXR and immersive web experiences
- Serverless architecture on AWS
- React Three Fiber and Three.js
- DevOps and infrastructure as code
- Blockchain and Web3 (when relevant)
- Creative technology intersections

## Output

Present ideas in a table format with actionable next steps.
