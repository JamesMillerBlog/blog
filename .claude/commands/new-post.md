# Create New Blog Post

Create a new blog post for jamesmiller.blog.

## Instructions

1. Ask the user for the topic if not provided: $ARGUMENTS

2. Research the topic:
   - Check existing posts in `_posts/` to avoid duplication
   - Consider what unique angle James can bring (WebXR, Serverless, AWS expertise)

3. Create the post structure:
   - Generate a URL-friendly slug
   - Create frontmatter with today's date
   - Write engaging content with code examples where appropriate

4. Write the MDX file to `_posts/[slug].mdx`

5. Run the build to verify: `pnpm build`

6. Suggest social media content to promote the post

## Post Format

Follow the style of existing posts:
- Technical but accessible
- Include practical code examples
- Use clear headings (##, ###)
- First person, conversational tone

## After Creation

Offer to:
- Generate social media posts
- Suggest related internal links
- Create a cover image prompt for AI image generation
