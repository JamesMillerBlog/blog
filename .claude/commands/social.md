# Generate Social Media Content

Generate social media content to promote a blog post.

## Instructions

1. Identify the target post:
   - If $ARGUMENTS provided, find that post
   - Otherwise, ask which post to promote or use the most recent

2. Read the post content from `_posts/`

3. Generate promotional content:

### Twitter/X Thread (5-7 tweets)
- Hook tweet that grabs attention
- Key insights from the post
- Code snippet or visual if relevant
- Call to action with link

### LinkedIn Post
- Professional tone
- Focus on business/career value
- Include relevant hashtags
- Encourage engagement

### Standalone Tweets (3-5)
- Different angles on the content
- Can be scheduled over time
- Include hashtags

4. Save to `_social/[slug].md` for reference

## Hashtag Guidelines

Consider: #WebXR #Serverless #AWS #React #ThreeJS #DevOps #WebDev #JavaScript #TypeScript #Blockchain #Metaverse

## Output Format

Present the content in a copyable format that can be directly posted.
